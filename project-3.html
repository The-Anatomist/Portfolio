Project Three

A simple calculator built in Python using the eval() function.

Project Overview

This project is a calculator I originally built in Python, using the built-in eval() function to evaluate expressions typed in by the user. The live demo below ports the same idea to JavaScript, safely evaluating the expression you build with the buttons.

Source Code Snippet

def calculate(expression): try: return eval(expression) except Exception: return "Error" expression = input("Enter expression: ") print(calculate(expression))
