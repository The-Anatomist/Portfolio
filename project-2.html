Project Two

A Rock, Paper, Scissors game — Python logic ported to JavaScript.

Project Overview

This project started as a simple Python script that let you play Rock, Paper, Scissors against the computer. I later ported the same decision logic to JavaScript so it could run live, right here in the browser.

Source Code Snippet

def determine_winner(player, ai): if player == ai: return "draw" beats = {"rock": "scissors", "paper": "rock", "scissors": "paper"} return "win" if beats[player] == ai else "lose"
